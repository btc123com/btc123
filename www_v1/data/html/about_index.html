BTC123 比特币/莱特币价格指数 是根据有影响力的BTC/LTC交易所最新价格，综合考虑各个因素的权重计算出来的比特币/莱特币价格指数，意在反映当前市场中比特币/莱特币综合价格。

比特币指数交易所选择：


挑选交易量较大，知名度较高的国内外交易所，目前共六家（后续会根据情况做出调整）。其中：
国外三家：Bitstamp, BTC-e, Bitfinex
国内三家：火币网, OKCoin中国版, 比特币中国

比特币指数权重计算：


综合考虑了各个交易所的市场深度，是否有交易手续费，是否有杠杆交易以及交易所的影响力，声誉等因素，设定各个交易所权重如下：
交易所权重
Bitstamp27
Bitfinex23
BTC-e15
火币网18
OKCoin国内站17
比特币中国16

当交易所经营状况发生变更时，此权重值亦将做相应的调整。

比特币指数计算方法：


BTC/USD价格指数 = ( Bitstamp最新价格 x Bitstamp权重 + Bitfinex最新价格 x Bitfinex权重 + BTC-e最新价格 x BTC-e权重 ) / ( Bitstamp权重 + Bitfinex权重 + BTC-e权重 )
BTC/CNY价格指数 = ( 火币网最新价格 x 火币网权重 + OKCoin国内站最新价格 x OKCoin国内站权重 + 比特币中国最新价格 x 比特币中国权重 ) / ( 火币网权重 + OKCoin国内站权重 + 比特币中国权重 )
BTC综合指数 则是抓取所有六家交易所比特币最新价然后按照权重做平均，其中三家美元交易所的价格将会按照最新汇率先折算为人民币价格，然后再做加权平均。

综合考虑了各个交易所的市场深度，是否有交易手续费，是否有杠杆交易以及交易所的影响力，声誉等因素，设定各个交易所权重如下：

当交易所经营状况发生变更时，此权重值亦将做相应的调整。

莱特币指数计算方法：


LTC/USD价格指数 = ( Bitfinex最新价格 x Bitfinex权重 + BTC-e最新价格 x BTC-e权重 ) / ( Bitfinex权重 + BTC-e权重 )
LTC/CNY价格指数 = ( 火币网最新价格 x 火币网权重 + OKCoin国内站最新价格 x OKCoin国内站权重 + 比特币中国最新价格 x 比特币中国权重 ) / ( 火币网权重 + OKCoin国内站权重 + 比特币中国权重 )
